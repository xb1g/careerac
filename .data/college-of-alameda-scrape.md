## ASSIST Articulation Data

**ASSIST.org** (https://www.assist.org) is the official California statewide repository for transfer articulation between community colleges and UC/CSU/AICCU institutions.

### Access Notes

- The ASSIST website is a JavaScript-heavy SPA (Single Page Application)
- Direct URL access with institution IDs does not render course content without JS execution
- The institution slug `college-of-alameda` at ASSIST was found to resolve to institution ID 13, which displays **Merritt College** course data — this appears to be a system configuration issue where CoA's slug may share an ID with Merritt College, or the ID mapping is incorrect
- Accessing the full ASSIST articulation database requires a JavaScript-enabled browser session

### ASSIST URL Patterns (for reference)

```
# Institution IDs confirmed:
# Merritt College = 13
# Foothill College = 51
# Modesto Junior College = 52
# Mt. San Jacinto College = 53
# Santa Ana College = 14

# Transferable Courses (CSU):
https://assist.org/transfer/results?year=75&institution={ID}&type=CSUTC&view=transferability

# Transferable Courses (UC):
https://assist.org/transfer/results?year=75&institution={ID}&type=UCETC&view=transferability
```

### How to Access CoA Articulation on ASSIST

1. Go to https://www.assist.org
2. Select **College of Alameda** from the community college dropdown
3. Choose the receiving institution (e.g., UC Berkeley, a specific CSU campus)
4. Select the academic year
5. View agreements by major, department, or general education

## Data Gaps & Recommendations

1. **ASSIST Articulation:** The official articulation database requires a JS-enabled browser. For the most accurate and complete data, students and counselors should use https://www.assist.org directly, selecting "College of Alameda" as the sending institution.

2. **Course Catalog Extraction:** The CurriQunet-based catalog at https://coa.curriqunet.com/catalog cannot be scraped without a headless browser capable of JS execution. The PDF archives are not text-extractable with current tools.

3. **CoA Institution ID on ASSIST:** The slug `college-of-alameda` resolves to institution ID 13 which shows Merritt College content. This should be reported to ASSIST/Peralta IT as a potential configuration error.

4. **Next Steps for Complete Data:** Manually export or screenshot the ASSIST articulation pages for CoA → UC Berkeley and CoA → CSU, or use a browser automation tool with JS support to scrape the CurriQunet catalog.
